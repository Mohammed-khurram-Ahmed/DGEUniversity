# DGEUniversity

## Quick Architectural Highlights

* **Architecture:** Built using a highly scalable **Modular MVI (Model-View-Intent)** pattern to ensure clean Architecture separation of concerns.
* **UI Layer:** Developed using **Jetpack Compose** and **XML** as per task requirements.
* **Tech Stack & Best Practices:** Kotlin, Jetpack Compose and XML, Hilt, Coroutines & Flow, Retrofit, Room.
